## Requirements

- NPM
- Download and install MongoDB : https://www.mongodb.com/download-center/community

## Start Server

Run MongoDb :

On Windows, execute mongod.exe

Then, start the nodeJS server :

```
$ npm start
```

Server starts on localhost:3000.

Check http://localhost:3000
